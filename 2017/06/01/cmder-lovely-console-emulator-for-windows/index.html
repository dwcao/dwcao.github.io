Cmder: lovely console emulator for Windows

Thu, Jun 1, 2017 cmder, cmd

Command Description
Ctrl + Tab Swift the tab window:
Ctrl + 1 Swift to the first tab window;
Ctrl + n Swift to the $n_{st}$ tab window;
Ctrl + T Create a new tab window.
Ctrl + W Close the recent tab window.
Alt + Enter Fullscreen

Shortcut to open Cmder in a chosen folder

  1. Open a terminal as an Administrator
  2. Excute Cmder.exe /REGISTER ALL
